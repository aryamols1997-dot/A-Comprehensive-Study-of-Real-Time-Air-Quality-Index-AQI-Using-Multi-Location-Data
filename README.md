# A-Comprehensive-Study-of-Real-Time-Air-Quality-Index-AQI-Using-Multi-Location-Data
This project analyzes real-time Air Quality Index (AQI) data across multiple locations to understand pollutant levels and their variations. It provides valuable insights to identify key focus areas and supports informed policy-making through clear and effective data visualization.
source:https://www.data.gov.in/resource/real-time-air-quality-index-various-locations
The analysis shows that the overall AQI falls in the satisfacory (100-200) category in many regions. It identifies carbon monoxide PM2.5 as a major pollutant, with vehicular emissions being a key contributing factor. States like Punjab and Rajasthan show high AQI levels, while Sikkim has relatively cleaner air. Various factors such as pollutant concentration, number of cities, and type of pollutants influence AQI levels. The project was implemented using Python (Pandas, NumPy) for data analysis and Matplotlib, Seaborn, and Plotly for visualization
